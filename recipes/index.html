Food Around the World

A study of cuisines around the world by Diana Cai and Gabrielle Ehrlich

We gathered data for recipes from AllRecipes.com for the cuisines Chinese, Korean, Japanese, Thai, Vietnamese, Indian, Filipino, Italian, French, Greek, Hungarian, and Mexican.

In this visualization, given a cuisine and a recipe from that cuisine, we compute a similarity score for recipes other recipes we are comparing it to as well as an aggregate similarity score for a cuisine. We represent each cuisine as a cluster, where in the center, we have the selected recipe with similar recipes clustered around it. The closer the node is to the recipe of interest, the more similar it is.

Similarly for other cuisines, the length of the cuisine node from the center shows how close together the aggregate scores are. The length of a recipe in that cluster from the center denotes how similar the recipe is to the currently selected recipe. We allow the user to select which cuisines to display (compute the similarity score) as well as adjust similarity thresholds and the number of nodes to display. Mousing over a node will show additional details, such as the ingredients. Clicking on a node will display simple information in a tooltip. Visit the link below to give it a try!
